# somlar-tokens

Shared design tokens for somlar projects.

- Fonts
- Color palette
- Radius + shadows
- Motion timing
- **Glass / glow:** `--color-glass` for glassmorphism backgrounds (use with `backdrop-filter: blur(var(--blur-glass))`); `--shadow-glow` for subtle border glow or highlight effects
- **Spacing:** `--spacing-card` for card inner padding; `--spacing-grid` for grid gap between cards
- **Status:** `--color-status-active` for active-state indicators (e.g. status dot, badges)
